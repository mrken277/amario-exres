Ubuntu 18.04

Server Installation#

erxes gives you several guides and options for installing the erxes platform.

The first guide is the Quick Start installation guide where we provide a script that automates most of the installation process. The second guide is a comprehensive, Step by Step installation guide where developers can see exactly what is being installed and how. The Step by Step installation guide is also used where heavy resources are needed when using erxes. Additional components are installed during the Step by Step installation.

Most users will find the Quick start installation to be more than sufficient. You can always upgrade at any time.

Ubuntu 18.04

The two installation guides are for installing erxes on Ubuntu 18.04. They should work on any hosting provider that allows you to install Ubuntu 18.04.

Ubuntu 18.04 is currently the only officially supported Linux distro. Other distros such as Centos and Debian are not specifically supported, but some users are able to adapt the below commands to make erxes work on them.

Minimum required server resources#

Each hosting company has slightly different server configurations and set-ups. Hosting companies such as Linode.com, Digital Ocean and AWS EC2 offer many different server specifications and configuration options.

To install and properly run your erxes installation, you must meet these minimum requirements:

Minimum requirements for the Quick Start guide#

  • Ubuntu 18.04 installed and running
  • 1 CPU
  • 1GB RAM
  • ability to to SSH into the server

Minimum requirements for the Step by Step guide#

  • Ubuntu 18.04 installed and running
  • 2 CPUs
  • 4GB RAM
  • ability to to SSH into the server
Use one-click installs

You can use the Digital Ocean Marketplace and AWS Marketplace for an even easier set-up experience.

Please install erxes with either the Quick Start or Step by Step instruction guides.
